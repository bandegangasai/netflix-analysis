# Netflix Data Analysis Full Stack Project

## 🚀 Overview
This is a full-stack project that analyzes Netflix data using a Spring Boot backend and a modern frontend.

## 🛠 Tech Stack
- Backend: Spring Boot (Java)
- Frontend: React / Vite
- Database: H2 Database
- Tools: Maven, Git, GitHub

## 📊 Features
- Upload Netflix dataset
- Process and analyze data
- Display insights and statistics
- REST API integration

### Backend
```bash
mvn spring-boot:run
